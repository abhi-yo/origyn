# The 3 Hero Templates

## Template 1: Serene Hero

**Style:**
- Light, clean design
- Blurred background image
- Centered text and buttons
- Minimal navigation

**Best for:**
- Business websites
- SaaS products
- Professional services
- Agencies

**Features:**
- Background image with blur effect
- Two CTA buttons
- Responsive navigation
- Custom font (Instrument Serif for headline)

**What to customize:**
- Headline text
- Description text
- Button text and links
- Background image URL
- Logo/brand name

## Template 2: Serenity Hero

**Style:**
- Dark theme
- Purple gradient orbs
- Glowing button effect
- Animated background

**Best for:**
- Tech products
- Apps and software
- Creative agencies
- Modern brands

**Features:**
- Animated gradient orbs
- Custom glow button
- Mobile menu
- Smooth animations

**What to customize:**
- All text content
- Button links
- Purple colors (change to your brand)
- Navigation menu items

## Template 3: Warmth Hero

**Style:**
- Warm orange/amber colors
- Floating particle animation
- Nature-inspired
- Soft, welcoming feel

**Best for:**
- Food and restaurants
- Wellness brands
- Lifestyle products
- Creative services

**Features:**
- Canvas particle animation (50 particles)
- Warm color palette
- Blurred background
- Smooth hover effects

**What to customize:**
- Headlines and text
- Button links
- Warm colors (change to your palette)
- Background image
- Brand name

## How to Use

1. Go to [Templates](http://localhost:3001/templates)
2. Preview each template
3. Click "Get Code" on your choice
4. Copy the entire code
5. Paste into your project
6. Update the text and links

## Customization Tips

**Change text:**
Look for strings in quotes and replace them.

**Change colors:**
Find color classes like `bg-purple-500` or `text-orange-900` and replace with your brand colors.

**Change images:**
Find `backgroundImage` or `src` properties and update the URL.

**Change links:**
Find `href="#"` and replace with your actual page URLs.
