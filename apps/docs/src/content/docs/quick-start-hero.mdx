# Get a Hero Section in 5 Steps

## Step 1: Sign Up

1. Go to [localhost:3001](http://localhost:3001)
2. Click "Dashboard"
3. Create an account with your email
4. Verify your email

## Step 2: Get Pro Access

1. Click "Dashboard" in the menu
2. Click "Upgrade to Pro"
3. Complete payment
4. Now you can access all templates

## Step 3: Pick a Template

1. Go to [Templates](http://localhost:3001/templates)
2. See the 3 hero options:
   - **Serene** - Clean, light background
   - **Serenity** - Dark with purple glow  
   - **Warmth** - Warm colors, animated
3. Click "Preview" to see each one
4. Choose the one you like

## Step 4: Get the Code

1. Click "Get Code" on your template
2. Copy all the code
3. Done!

## Step 5: Use It

### If you have a Next.js website:
1. Create a new file: `components/hero.tsx`
2. Paste the code
3. Import it in your page: `import { Hero } from '@/components/hero'`
4. Add `<Hero />` where you want it

### If you use other tools:
1. Paste the code into your project
2. Change the text and colors
3. Ask a developer if you need help

## Change the Text

Find and replace:
- Headlines - the big text
- Description - smaller text below
- Button text
- Menu items
